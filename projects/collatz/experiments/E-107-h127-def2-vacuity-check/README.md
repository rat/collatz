# E-107 — H-127's Definition 2 (diagonal configuration) is vacuous as written

Related hypothesis: [`H-127-reducao-z-number-dicotomia-espectral-wcc.md`](../../hypotheses/H-127-reducao-z-number-dicotomia-espectral-wcc.md)

## Origin

While trying to advance H-127 (closing Lemma B's Step 6: "upgrade from
diagonal configuration to continuous segment", marked as "plausible but
not written"), I consulted the Fable. The response revealed a problem
prior to, and more serious than, the declared gap: Definition 2 (the
diagonal configuration), as written, **does not distinguish structured
ξ from generic ξ** — it's satisfied by almost any ξ, including ξ=1.

## What it tests

Definition 2 (H-127): ξ has a (γ,δ,η)-diagonal configuration of depth s
if at ≥(1-η)s of the scales t=1..s there exists some exponent a in the
window [γt−t^(2/3), γt+t^(2/3)] with ‖ξ·2^a/3^t‖ < δ — **without
requiring coherence between the a_t chosen at consecutive scales**.

This script measures, for ξ=1, small ξ (5,7), and random ~200-bit ξ:
(a) the literal fraction of scales with a "hit" (the definition as
written); (b) the longest "chainable continuous segment" — a run of
consecutive scales with monotonic exponents a_t and bounded increments
(B = ⌊log₂(1/2δ)⌋) — the object that would actually carry rigidity
content (analogous to the "abundance heuristic" already present in
H-127).

## Result

At δ=1/6 and δ=1/10: fraction of hits **0.97-0.997 for ALL tested ξ,
including ξ=1** — no difference between structured and random ξ. In
contrast, the longest chainable segment stays at 5-9 for every ξ,
consistent with the Θ(log s) predicted for generic ξ (log 300 ≈ 5.7) —
not longer for ξ=1 than for random ξ.

## Interpretation

Definition 2, without Step 6's upgrade (chaining), concludes something
that holds for almost every ξ — Lemma B, using this definition, excludes
nothing: a "diagonal configuration" is not evidence of structure, it's
typical behavior. The object with real content is the **chained
segment of length ≫ log ℓ** — and it's exactly this upgrade that Step 6
needs to provide, and which, by two independent arguments (see H-127),
does not close with the tools in play (Riesz product, Halász, the
tilted gaps model). See H-127 for the full analysis of why both
possible routes (stability in j; intra-j chaining) fail.
